Présentation

Opérateur de solutions de sécurité cloud et hybride, formé par Microsoft (via Simplon), je suis amené à effectuer des tâches liées à la virtualisation (Azure, VMWare, KVM, LXC...) mais également à entretenir le bon fonctionnement et assurer la sécurité des serveurs et des infrastructures.

Les Langages

Python

Python est un langage polyvalent, interprété et de haut niveau. Il est utilisé dans divers domaines tels que le développement web, l'analyse de données, l'IA et la cybersécurité.

Bash

Bash est un langage de script pour Unix/Linux, utilisé pour automatiser les tâches système. Il offre des structures de contrôle et une interaction avec des commandes système.

PowerShell

PowerShell est un langage de script Microsoft pour l'administration système. Il offre une syntaxe orientée objet et une infrastructure flexible.

Les Outils

Shinken, Zabbix ou Azure Monitor

Zabbix et Shinken sont des solutions open-source de supervision réseau, tandis qu'Azure Monitor est conçu pour les environnements Microsoft Azure.

Virtualisation

Solutions comme KVM, VMWare, VirtualBox, HyperV pour des environnements variés, des datacenters aux tests individuels.

Azure

Azure est la plateforme cloud de Microsoft pour construire, déployer et gérer des applications et des infrastructures.
